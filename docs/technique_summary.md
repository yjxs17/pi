# 机械臂控制技术总结

## 核心模块概览

### arm_control.py
- **位置控制**：极坐标 arm_polar(theta, r)
  - theta: 70°(竖直) ~ 270°(竖直向下)，参考 90°=正前方
  - r: 80mm(收回) ~ 140mm(伸出)
- **夹爪控制**：claw(value)
  - 5=全开, 245=全闭
  - 小球最佳力度：180
  - 长条最佳力度：235
- **舵机补偿**：
  - 响应延迟：~167ms/step
  - safe_claw()：间隔200ms > 167ms（确保到位）
  - clamp_with_deadzone()：过冲+回落消除死区

### 五阶段流程

```
搜索 → 靠近 → 蹲下 → 微调 → 抓取
       ↓
   搬运(防掉落5层) → 放置
```

### 防掉落5层机制

1. **夹爪维持** — transport_tick() 每2.5s加固
2. **IMU平衡** — dog.imu(1) 自动抗颠簸
3. **低重心** — translation('z', 75)
4. **慢步态** — gait_type("slow_trot")
5. **转弯补偿** — on_turning() 抵消离心力

## 重要概念

### 安全初始化 safe_init()
- 必须在程序开始调用
- 调用 reset() 后等1.5s
- 张开夹爪后再收回机械臂
- 防止上次异常退出导致的卡位

### 抓取确认 verify_grasp()
- **机械验证**：提拉15mm检测是否夹牢
- **视觉验证**：检查球是否还在原位（HSV检测）
- 两者都通过才算成功

### 搬运维持 transport_tick()
- 每帧调用一次
- 每2.5s发送一次 claw(200) 加固
- 防止舵机松弛导致物体滑落

## 常见错误

| 错误 | 原因 | 解决 |
|------|------|------|
| 夹爪无响应 | 舵机掉电或地址错误 | 检查 claw 参数范围 [5, 245] |
| 臂伸不出去 | arm_polar 参数越界 | theta >= 70, r <= 140 |
| 抓取容易弹飞 | claw 力度太高导致冲击 | 用 safe_claw() 渐进闭合 |
| 搬运容易掉球 | 缺少夹力维持 | 每2.5s 调用 transport_tick() |

## 性能指标

- **单次抓取** ≤ 5s
- **抓取成功率** ≥ 90%
- **搬运稳定性** — 直线 100%、转弯 95%
- **放置精度** ≤ 3cm
- **单轮总耗时** ≤ 90s
